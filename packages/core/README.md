# @game-lab/core

Ядро игрового движка, предоставляющее базовые классы и системы для создания игр.

## Основные компоненты

### GameData
Абстрактный класс для управления состоянием игры.

### Gameflow
Презентер, управляющий потоком игры и обработкой событий.

### Scene
Базовый класс сцены на основе PixiJS Container.

### SoundService
Сервис для управления звуками и музыкой.

## Использование

\`\`\`typescript
import { Gameflow, Scene, SoundService } from '@game-lab/core';
\`\`\`
